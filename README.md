# FamShare - Social Media DBMS Project

## Features
- User registration and login
- Feed page
- Create, edit, delete posts
- Photo and video uploads
- Likes and comments
- Follow/unfollow system
- User profiles
- Search users and posts
- AI tools using offline Ollama
- Admin dashboard

## Requirements
- Python 3.10+
- Flask
- SQLite
- requests
- python-dotenv
- Werkzeug
